## 技术亮点

### 1. 硬件级叠加
- 直接操作显卡 Overlay Plane
- 绕过整个图形栈（X11/Wayland）
- 绝对优先级显示

### 2. 零延迟渲染
- 直接写入显存
- 无上下文切换
- 60fps 刷新率

### 3. 高可靠性
- 桌面崩溃不影响 HUD
- 系统级守护进程
- 优雅的错误处理

### 4. 极简架构
- 最小依赖（libdrm + cairo + pango）
- 最小资源占用
- 最大控制权

## 使用示例

### 基本使用
```bash
# 编译
make

# 运行（需要root权限）
sudo ./linuxhud -t "Hello LinuxHUD"
```

### 高级使用
```bash
# 自定义参数
sudo ./linuxhud -t "System Status" \
                -x 100 -y 100 \
                -w 400 -h 150 \
                -a 180 \
                -s 28 \
                -c "00FF00" \
                -b "000000" \
                -v

# 守护进程模式
sudo ./linuxhud -d -t "Background Service"
```

### 系统服务
```bash
# 安装为系统服务
sudo make install

# 启用服务
sudo systemctl enable linuxhud
sudo systemctl start linuxhud
```
